# 「间 MA」细胞植物工坊界面升级提示词

版本：2026-05-13  
用途：指导 LearningCell x 3DCellForge 融合系统的界面升级与验收

## 0. 总体定位

把原本「侘寂艺术展览落地页」翻译成一个可用的生物 3D 生成产品界面，产品名暂定为：

> 「间 MA」细胞植物工坊

它不是展览页，也不是营销页，而是一张可操作的纸面工作台。用户进入后应立刻看到 3D 生物模型舞台、模型索引、生成工作流和说明面板；艺术感服务于操作秩序，不压过功能。

核心气质：

- 老米色纸面、墨黑正文、朱红印章、苔绿色操作状态。
- 大量留白，但不牺牲可用信息密度。
- 细线分隔代替厚重卡片，纸张纹理代替渐变装饰。
- 3D 模型是“作品”，模型列表是“标本索引”，生成面板是“工坊札记”，说明面板是“观察记录”。

必须保留：

- 原有 3D 舞台与旋转/复位控制。
- 原有模型列表与生成模型列表。
- 文本生成工作流、导入 GLB、加载 3DCellForge 样例。
- 模型说明、关键结构、趣味知识、分布信息。

不照搬：

- 不做艺术展落地页 hero。
- 不做 Selected Works 网格。
- 不做虚构展览日期、地址、票价。
- 不把功能入口藏到滚动页面下方。

## 1. 视觉语言

### 1.1 色彩

UI 外壳只使用四组颜色：

- 纸面：`#f5f1e8`、`#ede7d7`、`#e4dcc6`
- 墨黑：`#1c1c1a`、`#2b2a26`
- 静音灰：`#6a665c`、`#9a9487`
- 朱红：`#8b3a2e`
- 苔绿只用于成功状态、主行动按钮和生命感暗示：`#4a5d3a`

模型缩略图与 3D 模型本身允许保留生物结构色彩；除此之外不要使用紫色、蓝色、玫红、彩色渐变。

### 1.2 字体与层级

- 标题、模型名称、品牌字使用宋/明朝风格 serif。
- 功能标签、状态、编号、英文辅助信息使用 mono，字距拉开。
- 正文使用清晰 sans 或系统中文无衬线。
- 不用夸张超大营销标题；主舞台标题可大，但要像作品题签。

### 1.3 纸面与细线

- 全局背景有极淡纸纹噪点。
- 面板边界使用 1px hairline。
- 面板圆角控制在 8px 内。
- 减少厚阴影，保留轻微纸张层次。
- 用细线、编号、留白建立秩序。

## 2. 产品布局

桌面端保持三列，但重新解释：

```text
顶部：项目编号 / 品牌「间 MA」/ 当前能力状态
左列：模型索引 Specimen Index
中列：3D 主舞台 Observation Stage
右列：生成工坊 + 观察说明 + About 札记
底部：版权与当前版本号
```

### 2.1 顶栏

- 左侧：`N° 07 · CELL PLANT FORGE · 2026`
- 中间：品牌标记 `間 | MA CELL STUDIO`
- 右侧：`Generate / Specimens / About / Local API`
- 顶栏下方必须有横向 hairline。
- 加一个朱红小印章，文字为「間」，像盖在纸面右侧。

### 2.2 左侧模型列表

- 标题改为 mono 标签：`SPECIMEN INDEX / 标本索引`
- 官方模型和生成模型分成两段。
- 每个模型像纸面上的标本条目：小缩略图、名称、分类/来源、状态。
- 激活态使用苔绿细边与浅纸色，不使用大色块渐变。

### 2.3 中央 3D 舞台

- 保持最大视觉权重。
- 舞台背景像一张浅纸台，边界用细线。
- 3D 模型背后放一个极淡圆相/墨晕，不能遮挡模型。
- 左上保留模型名称与副标题，但改成展签式排版。
- 角落加入 mono 编号，例如 `§ 01 — Observation Stage`。
- 保留自动旋转与复位视角按钮，按钮改成细线纸面按钮。

### 2.4 右侧生成与说明

- 生成面板题名改为“生成工坊”，保留文本输入、模板选择、provider、三个操作按钮、状态、历史任务。
- 说明面板题名保持生物学信息，但视觉上改成“观察记录”。
- 新增一个轻量 About 札记，放在右侧底部：
  - 解释「间 MA」在本项目里是“在描述、模型与观察之间保留可生成的空白”。
  - 保留圆相、mono 标签和朱印。

## 3. 交互与动效

- 不做复杂落地页滚动动画，避免影响产品操作。
- 页面进入时只做轻微淡入。
- 朱印可有轻微盖章感。
- 按钮 hover 只做细线加深或轻微上移。
- 任务进度条使用苔绿线，不使用蓝紫渐变。
- 支持 `prefers-reduced-motion`，减少动效。

## 4. 响应式

桌面端强调三列工作台。  
平板端压缩左右列宽，保持中央舞台优先。  
移动端改为：

```text
顶栏
横向模型索引
3D 舞台
生成工坊
观察记录
About 札记
```

移动端允许滚动，不能隐藏生成工作流和生成模型列表。
